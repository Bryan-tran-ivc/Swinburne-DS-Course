# Super Mario Bros Clone

A Pygame-based Super Mario Bros clone (World 1-1) written in Python.

---

## Requirements

Before running the game, make sure you have the following libraries installed:

- **Pygame** — handles graphics, sound, and input
- **pytmx** — loads the tile map file for the game world

```bash
pip install pygame pytmx
```

---

## Running the Game

```bash
python main.py
```

---

## Controls

| Key | Action |
|-----|--------|
| ← → | Move left / right |
| Enter | Confirm on menus |
